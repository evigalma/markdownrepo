<h4>安裝</h4>

1. 安裝Git:[Git網站連結](https://git-scm.com/downloads")，依使用者的作業系統選擇下載安裝檔，並執行安裝程序。<br>

2. 建立一個Bitbucket帳號:[Bitbucket網站連結]("https://bitbucket.org/")

3. 安裝SourceTree:[SourceTree網站連結]("https://www.sourcetreeapp.com/download-archives")，依使用者的作業系統選擇下載安裝檔，並執行安裝程序。<br>


<h4>環境設定</h4>

1. 設定global ID password

2. ssh金鑰--SSH Key



<h4>建立一個Repo</h4>

**Git:使用Git Bash**

<h4></h4>

1. Init : 在要進行版本控管的資料夾輸入指令建立git倉儲
```git
$ git init
```

2. Clone
```git
$ git clone <repo URL>
```

**SourceTree**

1. 使用Create在本機建立一個有git倉儲的資料夾

2. 使用Add，針對已建立的資料夾加入git倉儲

3. Clone，將來自Github或Gitlab已經建立好的Repo下載至本機建立一個資料夾
